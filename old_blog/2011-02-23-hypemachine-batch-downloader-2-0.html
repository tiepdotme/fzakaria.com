HypeMachine Batch Downloader 2.0

Published 2011-02-23 on Farid Zakaria's Blog — Permalink

They're going to hate me..

Going to keep this one short and sweet.

So yesterday I decided to rewrite and fix the batch downloader script I had written in Python to download songs from HypeMachine. Anyways, you can find the find the new script and accompanying README file here (or the Projects page). The script should be straight forward to setup and run. If you have any issues please let me know.

EDIT: If you would like the script feel free to send me an e-mail. I removed the link from my blog upon request of the site creators of the website.

More to come

If there seems to be a demand, I might port it to some GUI form, to make it even simpler to get the songs. In the mean time I plan on fixing the HypeMachine chrome extension to allow browsing and playback at the same time. I also plan on porting the extension to Safari and Firefox.

Enjoy the songs. =)
